## 📕书籍推荐(初级版)

### 📖 《富爸爸，穷爸爸》

- 作者：罗伯特·T·清崎
- 豆瓣评分：8.3分
- 下载地址：[PDF版下载地址](https://pan.quark.cn/s/ca96f4cf0440)（**先转存再查看，以防失效**）

世界上最畅销的理财书。该书被翻译成了 50 多种语言，卖出几千万册。通过这本书能直观的感受的“富人思维”和“穷人思维”的区别。帮助我们对现金流、资产建立基本的认知。

#### 全书梗概

>为啥有人忙到死赚不到大钱，有人躺着也能收钱？这本书就讲两个爸爸的财富真相：
>
>穷爸爸是高材生，一辈子上班攒钱，觉得 “稳定最重要”，最后越忙越穷；富爸爸没高学历，却懂让钱干活，教作者 “别为钱打工，要让钱为你打工”。
>
>核心就 3 件事：
>
>① 分清资产（能自动给你送钱的，比如收租的房子）和负债（要你不断掏钱的，比如只开不赚的车）；
>
>② 财商比学历管用，要学赚钱的规则，而不是只学找工作的技能；
>
>③ 别怕风险，学会驾驭风险，才能跳出 “上班 - 赚钱 - 花钱” 的死循环。
>
>看完就懂：
>
>原来变富不是靠攒工资，而是靠打造 “自动赚钱机器”，普通人也能学会的财富逻辑，越看越想立刻行动！

### 📖《小狗钱钱》

- 作者： [德\] 博多·舍费尔
- 豆瓣评分：9.2分
- 下载地址：[PDF版下载地址](https://pan.quark.cn/s/a1913082a489)（**先转存再查看，以防失效**）

欧洲最畅销的理财图书。深入浅出，通俗易懂。讲解如何从小学会支配金钱，而不是受金钱的支配；如何像富人那样思考，正确地认识和使用金钱；如何进行理财投资，找到积累财富的正确方法，并早日实现财务自由。

#### 全书梗概

>想存钱总忍不住花？想赚钱不知道从哪下手？这本超萌的书，让一只会说话的小狗教你变有钱！
>
>普通小女孩和小狗钱钱相遇后，学会了 3 个超简单的财富魔法：
>
>① 列 3 个最想要的梦想（比如去迪士尼），专门存钱；
>
>② 收入分 3 份：存起来、花一点、帮别人；
>
>③ 养 “会生钱的鹅”（比如存钱赚利息），鹅越大，钱越多。
>
>没有复杂公式，全是小孩都能懂的大白话，跟着做就能告别月光，慢慢攒下属于自己的 “小金库”，还能学会让钱越变越多！看完就想立刻列梦想、存钱，原来变富一点都不难～
